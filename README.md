# Rakshu_chatUI

A modern responsive Chat UI assignment project built with:

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap 5

## Features

- Welcome screen with suggestion cards
- User and assistant message bubbles
- Typing indicator with animated dots
- Auto-resizing textarea
- Enter to send / Shift+Enter new line
- Sidebar with mobile slide-in menu
- Fully responsive (mobile, tablet, desktop)


## Folder Structure

- `index.html`
- `css/style.css`
- `js/chat.js`
- `screenshots/` (add desktop/mobile/tablet screenshots)

- ## Run

1. Open `index.html` in browser  
or  
2. Use VS Code Live Server for best experience.
